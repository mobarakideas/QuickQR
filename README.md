# ⚡ QuickQR – Smart QR Code Generator

QuickQR is a powerful and user-friendly web application that allows users to generate, download, and manage QR codes easily. It supports storing QR history per user with login authentication.

---

## 🚀 Features

- 🔐 User Authentication (Login/Register)
- 📱 Generate QR Codes (Text, Contact, Business Info, etc.)
- 📥 Download QR Codes
- 🗂️ User-specific QR History
- ✏️ Edit / Delete QR Codes
- 🎨 Modern UI with animations
- ⚡ Fast and responsive design

---

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, Bootstrap, JavaScript  
- **Backend:** PHP  
- **Database:** MySQL  
- **Server:** XAMPP
